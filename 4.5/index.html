Introduction

Welcome to 51Degrees

51Degrees helps you enrich user data in real time by detecting device details, geolocation, and IP Intelligence. Our tools power high-performance web experiences, fraud prevention, targeted content delivery, and more.

We offer both cloud-hosted and on-premise solutions, with support for major programming languages.



Get Started Fast

Choose a service - or combine them together:

Device Detection

Identify device types, capabilities, and features to optimize experiences across phones, tablets, TVs, and more.

  • Start by reading the Device Detection Overview to understand how it works and its benefits.
  • Follow the Device Detection Quickstart to add it to your application.
  • Learn how to detect Apple models with Apple Device Detection using client-side JavaScript.

IP Intelligence

Gain insights into user locations and IP addresses for targeted content, analytics, and fraud prevention.

  • Read the IP Intelligence Overview to learn about our IP intelligence capabilities.
  • Follow the IP Intelligence Quickstart to integrate IP intelligence into your application.

Reverse Geocoding

Obtain accurate real-world location data from coordinates, such as postal addresses, using our Reverse Geocoding service.

  • Learn how it works in the Reverse Geocoding Overview.
  • Follow the Reverse Geocoding Quickstart to implement reverse geocoding in your application.

Client-side JavaScript

If you're working entirely in the browser, you can use 51Degrees without server-side code.

Use the Cloud Configurator to select the properties you want and generate a custom JavaScript snippet to embed in your page.

See the Configurator explanation for instructions on how to obtain a Resource Key and use it with the generated snippet.



Supported Languages

We offer official libraries for:

.NET (C#)

  • device-detection-dotnet - for Device Detection
  • device-detection-dotnet-examples - examples for Device Detection
  • ip-intelligence-dotnet - for IP intelligence
  • ip-intelligence-dotnet-examples - examples for IP intelligence
  • location-dotnet - for Reverse Geocoding
  • pipeline-dotnet - for the Pipeline API

Java

  • device-detection-java - for Device Detection
  • device-detection-java-examples - examples for Device Detection
  • ip-intelligence-java - for IP intelligence
  • ip-intelligence-java-examples - examples for IP intelligence
  • location-java - for Reverse Geocoding
  • pipeline-java - for the Pipeline API

Node.js

  • device-detection-node - for Device Detection (with examples)
  • location-node - for Reverse Geocoding
  • pipeline-node - for the Pipeline API

Go

  • device-detection-go - for Device Detection
  • device-detection-examples-go - examples for Device Detection

Python

  • device-detection-python - for Device Detection (with examples)
  • location-python - for Reverse Geocoding
  • pipeline-python - for the Pipeline API

PHP

  • device-detection-php - for Device Detection (with examples)
  • device-detection-php-onpremise - for Device Detection with on-premise data (with examples)
  • location-php - for Reverse Geocoding

Client-side JavaScript

  • Check out our Configurator page for details on how to use client-side JavaScript with Device Detection.

Other Languages / Tools

  • device-detection-nginx - for Device Detection in Nginx
  • device-detection-varnish - for Device Detection in Varnish
  • UA Parser JS - our fork of the popular UA Parser library, with support of UA-CH
  • Device Detection Prebid Integration - for integrating Device Detection with Prebid.js/Prebid Server
