Aspect Properties

Introduction

Aspect properties extend element properties to add metadata relevant to Aspect Engines.

As Aspect Engines consume external data from either a data file or Cloud Service, extra metadata is included to indicate where the property is available.

See the Specification for more technical details.

Data tier

The data tier of a property describes which product tiers the property is available in. For example, an Aspect Engine could consume both a free and enterprise data file. Please be aware that certain properties may not be available in the free data file..
