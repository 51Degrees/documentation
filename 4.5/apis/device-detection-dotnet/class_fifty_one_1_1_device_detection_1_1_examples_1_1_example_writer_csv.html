FiftyOne.DeviceDetection.Examples.ExampleWriterCsv Class Reference

Detailed Description

Simple writer to a text stream in CSV format.

Must be disposed of to write the records. As CSV requires the headings to be known in advance this can not be done until all the records have been passed to the writer.
