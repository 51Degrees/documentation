FiftyOne.IpIntelligence.Cloud.Data.CloudCountriesTranslationData Class Reference

Detailed Description

Cloud counterpart of the on-premise FiftyOne.IpIntelligence.Translation.Data.CountriesTranslationData.

Implements the same ICountriesTranslationData contract so consumers see an identical surface whether the data came from the cloud service or an on-premise pipeline. Populated by FlowElements.CloudCountriesTranslationEngine.
