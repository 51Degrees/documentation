FiftyOne.IpIntelligence.Engine.OnPremise.Data.ValueMetaData Class Reference

Detailed Description

Data class that contains meta-data relating to a specific value.

This implementation of the IValueMetaData interface is used for meta-data that is generated by the native C/C++ code from information in the data file.
