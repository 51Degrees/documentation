The evidence data from this event.

The dictionary key is the first part of the evidence key. The value is another dictionary containing the rest of the evidence key and the evidence value. For example, the evidence "header.user-agent"="ABC123" would become:

{ Key = "header", Value = { Key = "user-agent" Value = "ABC123" }}
