TranslationEngineBuilder FiftyOne.Pipeline.Translation.FlowElements.TranslationEngineBuilder.AddSource ( string filePath)

Add a source file containing translations.

These follow the naming convention 'abc.en_GB.yml' where 'abc' can be any idenitifier, 'en_GB' is the locale code, and 'yml' is the file extension. The locale code is used to determine which language is contained in the translation files. Files must be in YAML format. The source can contain a wildcard to add multiple files e.g. 'abc.*.yml' to add all languages for the 'abc' identifier.

Parameters
filePathThe path to the source file where a wildcard can be used for the language component of the filename.
Returns
This builder.

Set a fixed language for the translation engine to translate to.

If this is set, the engine will not determine the language to translate to from the source element data, it will instead always translate to this language.
