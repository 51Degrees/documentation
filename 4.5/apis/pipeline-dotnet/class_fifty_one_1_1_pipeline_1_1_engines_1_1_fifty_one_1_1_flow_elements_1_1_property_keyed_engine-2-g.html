FiftyOne.Pipeline.Engines.FiftyOne.FlowElements.PropertyKeyedEngine< TData, TProfile > Class Template Referenceabstract

Detailed Description

Base class for engines that use an index of property values to profile ids to populate one or more profile instances if a valid value for a specific configured property is provided in the evidence.

The specific property to look for is determined by GetKeyPropertyName().

Template Parameters
TDataThe type of multi-profile data returned by the engine. Must implement IMultiProfileData<TProfile>.
TProfileThe type of profile data contained in the multi-profile result. Must implement IAspectData.
