MissingPropertyResult FiftyOne.Pipeline.Engines.Services.IMissingPropertyService.GetMissingPropertyReason ( string propertyName,
IReadOnlyList< IAspectEngine > engines,
IAspectData aspectData )

Get the reason that the specified property is not available in the results from the specified engines, with access to the aspect data instance so that per-request state (e.g.

a cloud-request-failed marker) can be considered.

Cloud aspect data carries a per-request marker that flips when the upstream cloud call failed. The on-premise service ignores this overload's aspectData argument; the cloud service uses it to short-circuit to MissingPropertyReason.CloudRequestFailed without mis-reporting via the standard engine-metadata heuristics.

Parameters
propertyNameThe property name to check.
enginesThe engines that were expected to populate the property.
aspectDataThe aspect data instance the missing property was requested from. May be null if the caller has no per-request context.
Returns
A MissingPropertyResult instance explaining why the property is not populated.
