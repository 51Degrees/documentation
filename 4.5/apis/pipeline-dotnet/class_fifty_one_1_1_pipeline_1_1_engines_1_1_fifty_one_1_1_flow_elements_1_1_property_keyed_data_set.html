FiftyOne.Pipeline.Engines.FiftyOne.FlowElements.PropertyKeyedDataSet Class Reference

Detailed Description

Contains the property indexes, pipeline, and inner engine used by PropertyKeyedEngine<TData, TProfile> to look up profile ids from property values.
