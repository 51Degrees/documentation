FiftyOne.Pipeline.Translation.Data.Translator Class Reference

Detailed Description

Translator used to translate values based on a set of translations.

The translator supports translating various string types. The result is the same type as the source e.g. a string will be translated to a string, a list of strings will be translated to a list of strings, etc.

object FiftyOne.Pipeline.Translation.Data.Translator.Translate ( object value,
IList< Exception > errors )

Translate the object to the language this translator is configured for.

The translator supports translating various string types. The result is the same type as the source e.g. a string will be translated to a string, a list of strings will be translated to a list of strings, etc. For AspectPropertyValue types, if the value has no value, then the no value message will be copied to the result.
