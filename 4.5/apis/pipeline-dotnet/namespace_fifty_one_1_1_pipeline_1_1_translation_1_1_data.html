enum FiftyOne.Pipeline.Translation.Data.MissingTranslationBehavior
strong

Defines the behavior of the translation engine when a translation is missing for a value.

Enumerator

Original 

Use the original value if there is no translation for it.

This is the default behavior.

EmptyString 

Use an empty string if there is no translation for the value.

FlowError 

Add a flow error if there is no translation for the value.

The error contains the reason there was no translation.
