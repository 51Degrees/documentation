int fiftyone.ipintelligence.examples.web.EmbedJetty.boundPort ( Server server)
static

Returns the local TCP port the server's first connector is actually listening on.

When the server is started with port 0 the OS assigns a free ephemeral port; tests should read it back via this method rather than assuming a fixed port.
