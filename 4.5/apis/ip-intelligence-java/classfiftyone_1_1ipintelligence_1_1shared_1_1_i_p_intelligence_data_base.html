◆ getAccuracyRadiusMax()

AspectPropertyValue< Integer > fiftyone.ipintelligence.shared.IPIntelligenceDataBase.getAccuracyRadiusMax ( )

Radius in kilometers of the circle centred around the most probable location that encompasses the entire area.

Where multiple areas are returned, this will only cover the area the most probable location is in. See Areas property. This will likely be a very large distance. It is recommend to use the AccuracyRadiusMin property.

Implements fiftyone.ipintelligence.shared.IPIntelligenceData.
