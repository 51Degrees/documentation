fiftyone.ipintelligence.examples.console.GettingStartedMixed Class Reference

Detailed Description

Demonstrates using both Device Detection and IP Intelligence engines in a single pipeline.

Both engines process the same evidence in parallel, providing comprehensive device and location information from a single pipeline invocation.

This example uses a pipeline configuration file that configures both engines. The configuration file is src/main/resources/gettingStartedMixed.xml.
