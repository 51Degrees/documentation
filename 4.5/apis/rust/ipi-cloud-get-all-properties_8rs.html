◆ render_value()

fn render_value ( value:& PropertyValue) -> String

Render any property value for display.

IP-intelligence properties are weighted, so in the dynamic bag they appear as a KeyValueList of {value, weight} records. This renders each record as value (weighting w) so the weighting is visible, and falls back to the shared string rendering for any non-weighted property the key might also return. Long results are truncated.
