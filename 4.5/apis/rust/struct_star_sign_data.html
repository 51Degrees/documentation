StarSignData Struct Reference

Detailed Description

The element data produced by every star-sign example.

It embeds a [MapElementData] so it gets the dynamic property bag (and the case-insensitive get/keys) for free, exactly as the core's DataBase pattern intends. The strongly-typed [StarSignData::star_sign] and [StarSignData::dob_javascript] accessors read back the two properties the examples populate.
