To get started with C IP intelligence on-premise:

  1. Clone the GitHub repository.
  2. Ensure you have Git LFS installed. The device data files are large binary files that can cause problems if stored in a Git repository directly so Git LFS is used.
  3. Ensure all submodules are checked out by running git submodule update --init --recursive in the repository.
  4. Follow the installation instructions to get set up with the project.
  5. Follow the linked example here: Getting Started.
  6. (optional) Obtain a license key by starting a free trial and download a data file with access to more properties, see our pricing page for details.
