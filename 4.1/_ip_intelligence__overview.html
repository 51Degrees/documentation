IP Intelligence - Overview

Introduction

By providing various data associated with an IP address, IP intelligence enables you to customize and provide users with the best online experience in a privacy sensitive manner.

If that excites you, let start with trying IP intelligence by following the quick start guide. The guide is straightforward and can get you up and running with just a few clicks. If you would like to know more about other great features that IP intelligence can deliver, then carry on reading below.

How IP Intelligence Works

51Degrees IP intelligence simply takes an IP address and returns the data that matches that IP address such as locations, Internet service providers, type of connectivity and so much more.

Weighted Data

Data being returned for each IP address will be accompanied with their weights, such that users will not get one single answer. Instead, users will get a list of results in which, each result will have a percentage to indicate how likelihood of it being the right match. This list is provided in descending order so the first result will always be the one with the highest percentage.

IP related data are volatile and by providing their weights, you will have the flexibility to implement strategy which will cover various scenarios, ensuring users experience will always be at their highest expected.

Location Bound

Location is best provided with an estimated area rather than a weighted list. When requesting location for an IP address, users can choose the possible properties which are the average coordinate and its bound South East and North West corners. Using these coordinates, you can easily map the potential area of the IP address and have a more comprehensive view of the data that can be within the range.
