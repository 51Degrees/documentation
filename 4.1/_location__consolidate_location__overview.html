ConsolidateLocation - Overview

Introduction

The Consolidate Location Engine takes results from multiple geo-location sources such as the Reverse Geocoding and the IP Intelligence Engine and combines the results into a single output.

This makes is incredibly easy to consume multiple location products and more importantly, you can change the location provider without modifying your code. For example if you are using 51Degrees cloud location but then whish to switch to the 3rd party Digital Element addon, only your [resource key]() will need to be updated.

If in case you are comsuming results from multiple providers, results for reverse geo location providers will take precedence over IP Intelligence results as reverse geo location results are obtained from a latitude and longitude which will be more accurate than a location obtained from an IP Address.

There is a quickstart guide to get you up and running ASAP or you can browse the more in-depth examples first.
