SimpleClientSideElement Struct Reference

Detailed Description

The client-side star-sign element.

Unlike the simple element, this one does not expect the birth date to be supplied directly. On the first request there is no cookie.date-of-birth, so it cannot compute a sign; instead it emits a JavaScript snippet (the dobjavascript property) that prompts the visitor, writes their answer to the cookie and reloads the page. The 51Degrees client JavaScript bundle runs that snippet on the client. On the reload the cookie is present, the element computes the sign, and the JavaScript becomes empty.
