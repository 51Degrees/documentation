◆ FindDataFile()

string FiftyOne.DeviceDetection.Examples.ExampleUtils.FindDataFile ( string filename,
DirectoryInfo dir = null )
static

Get the path to a device detection data file.

The environment variables are checked first for an explicit path. The aligned variable named by Constants.DEVICE_DETECTION_DATA_FILE_ENV_VAR takes precedence over the legacy variable named by Constants.LEGACY_DEVICE_DETECTION_DATA_FILE_ENV_VAR. If neither is set, the folder hierarchy is searched for the supplied file name using FindFile(string, DirectoryInfo).

Parameters
filenameThe data file name to search for when no environment variable supplies an explicit path.
dirThe directory to start searching from. If not provided the current directory is used.
Returns
The path to the data file, or null if no file could be found.
Examples
OnPremise/Framework-Web/Default.aspx.cs, OnPremise/PerformanceMultiProperty-Console/Program.cs, and OnPremise/PerformanceSweep-Console/Program.cs.

DirectoryInfo dir = null )

Uses a background task to search for the specified filename within the working directory.

If the file cannot be found, the algorithm will move to the parent directory and repeat the process. This continues until the file is found or a timeout is triggered.

Parameters

dirThe directory to start looking from. If not provided the current directory is used.
Returns
