FiftyOne.Pipeline.Engines.Services.MissingPropertyService Class Reference

Detailed Description

Service that determines the reason for a property not being populated by an engine.

See the Specification

Cloud-specific behaviour that depends on per-request state — such as reporting MissingPropertyReason.CloudRequestFailed when the upstream cloud request failed for the request that produced the aspect data — is added by the cloud-specific subclass MissingPropertyServiceCloud in the CloudRequestEngine package.
