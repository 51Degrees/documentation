string FiftyOne.Pipeline.Engines.FiftyOne.Data.FiftyOneDataFile.DataDownloadType
getset

Get the type name to send when checking for data file updates.

In general, this value should be pulled from the Engine, which will have read it from the data file. However, in some cases, we want to know the type name before the engine is created. (e.g. when UpdateOnStartup is set) This is why the value can also be set.
