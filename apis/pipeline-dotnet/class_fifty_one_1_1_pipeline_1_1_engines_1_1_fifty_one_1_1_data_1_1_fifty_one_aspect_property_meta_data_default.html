FiftyOne.Pipeline.Engines.FiftyOne.Data.FiftyOneAspectPropertyMetaDataDefault Class Reference

Detailed Description

Data class that contains meta-data relating to a specific property.

This variation is used for meta-data that is generated by the c# code rather than the native C/C++ code. For example, this is used to add placeholder property meta-data for the 'metrics' properties because they don't have any associated meta-data in the data file.
