FiftyOne.Pipeline.Core.FlowElements.IFlowElement< T, TMeta > Interface Template Reference

Detailed Description

Generic interface that extends IFlowElement with the ability to return a ITypedKey<T> that will allow type-safe access to the data that this element populates in IFlowData.

Template Parameters
TThe type of element data that the flow element will write to IFlowData.
TMetaThe type of meta data that the flow element will supply about the properties it populates.
Type Constraints

TMeta :IElementPropertyMetaData
