FiftyOne.IpIntelligence.Engine.OnPremise.Data.ComponentMetaData Class Reference

Detailed Description

Data class that contains meta-data relating to a specific component.

A component is a logical group of properties for example, all properties relating to the autonomous system number or all properties relating to the internet service provider. This variation is used for meta-data that is generated by the native C/C++ code using the information in the data file.
