FiftyOne.IpIntelligence.Cloud.FlowElements.CloudCountriesTranslationEngine Class Reference

Detailed Description

Cloud counterpart of the on-premise FiftyOne.IpIntelligence.Translation.FlowElements.CountriesTranslationEngine.

Reads the already-computed country name translations from the cloud service response (the cloud runs the translation pipeline server-side) and exposes them through ICountriesTranslationData. Like IpiCloudEngine, its property list is driven by the cloud metadata, so a resource key that does not grant the translation properties fails fast when the pipeline is built.

Perform the processing for this engine:

  1. Extract the "countrynamestranslated" section from the cloud JSON.
  2. Deserialize it to populate a CloudCountriesTranslationData instance.

The base CloudAspectEngineBase<T> handles fetching the JSON from the CloudRequestEngine, the missing-engine error and the empty-response (cloud request failed) case, then calls this method with the JSON.

Parameters
data - The IFlowData instance containing data for the current request.
aspectData - The CloudCountriesTranslationData instance to populate.
json - The JSON response from the CloudRequestEngine.
