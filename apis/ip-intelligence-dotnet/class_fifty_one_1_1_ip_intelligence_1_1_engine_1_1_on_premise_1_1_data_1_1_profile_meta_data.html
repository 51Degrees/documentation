FiftyOne.IpIntelligence.Engine.OnPremise.Data.ProfileMetaData Class Reference

Detailed Description

Data class that contains meta-data relating to a specific profile.

A profile is a set of specific property values for an individual component. This implementation of the IProfileMetaData interface is used for meta-data that is generated by the native C/C++ code from information in the data file.
