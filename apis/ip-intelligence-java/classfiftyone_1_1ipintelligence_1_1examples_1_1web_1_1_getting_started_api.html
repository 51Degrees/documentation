fiftyone.ipintelligence.examples.web.GettingStartedApi Class Reference

Detailed Description

Provides a local HTTP API for IP Intelligence lookups, mirroring the endpoints of the 51Degrees Cloud service.

The pipeline contains the on-premise IP Intelligence engine and a JSON builder element which serialises the results.

The data file can be supplied as a command line argument. When no argument is given the enterprise data file is looked for in the ip-intelligence-data directory.

Note that the .NET version of this example also includes a device detection engine in the same pipeline. This Java version is IP Intelligence only.
