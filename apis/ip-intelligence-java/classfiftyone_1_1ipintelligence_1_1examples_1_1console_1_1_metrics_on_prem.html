fiftyone.ipintelligence.examples.console.MetricsOnPrem Class Reference

Detailed Description

Iterates over the valid IP ranges contained in the data file, samples IP addresses from each range, and records metrics grouped by continent, country, location confidence, connection type and network flags.

For each group the number of IP addresses, the average geographic area in square kilometers, the equivalent circle radius and the average number of polygons are output in CSV format.
