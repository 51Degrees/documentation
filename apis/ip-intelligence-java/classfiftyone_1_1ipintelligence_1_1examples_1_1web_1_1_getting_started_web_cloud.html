fiftyone.ipintelligence.examples.web.GettingStartedWebCloud Class Reference

Detailed Description

This example shows how to use 51Degrees Cloud IP Intelligence to determine location and network details from IP addresses in a web application.

This is the getting started Web/Cloud example showing use of the 51Degrees supplied filter which automatically creates and configures an IP Intelligence pipeline.

The configuration file for the pipeline is at src/main/webapp/WEB-INF/51Degrees-Cloud.xml. The resource key is taken from the environment variable or system property "TestResourceKey" unless supplied as a command line argument.
